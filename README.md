# 🎬 Netflix Movie Data Analysis (Python | EDA)

This project is a complete exploratory data analysis (EDA) of Netflix movie data using Python. The dataset includes 9,000+ movies with details like genre, popularity, votes, and release year.

## 📌 Key Questions Answered:
1. What is the most frequent genre of movies released on Netflix?
2. Which movie has the highest average votes?
3. What movie got the highest popularity and what is its genre?
4. What movie got the lowest popularity and what is its genre?
5. Which year had the most movie releases?

## 🧰 Tools & Libraries:
- Python (Jupyter Notebook)
- Pandas, NumPy
- Matplotlib, Seaborn

## 📁 Files Included:
- `Movie_data_analysis_netflix.ipynb` – Main analysis notebook
- `netflix_movies.csv` – Dataset used for analysis
